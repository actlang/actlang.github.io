ActLang FAQs

How is ActLang different from other agent frameworks?

Other agentic frameworks can work for simple, generic tasks but fall short for complex tasks bespoke to a companys needs. ActLang provides a more expressive framework to handle companies unique tasks without restricting users to a single black-box cognitive architecture.

Does ActLang impact the performance of my app?

ActLang will not add any overhead to your code and is specifically designed with streaming workflows in mind.

Is ActLang open source? Is it free?

Yes. ActLang is an MIT-licensed open-source library and is free to use.
